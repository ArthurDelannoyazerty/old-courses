Projet ISN

Pour ce projet, nous avons l'intention de créer un programme sous java qui a pour but d'utiliser l'ordinateur sans la souris mais seulement avec le regard. Pour cela, nous allons utiliser une librairie/API java nommée : OpenCV. Elle permet de traiter plus facilement des images, ce qui est plus pratique pour nous.
